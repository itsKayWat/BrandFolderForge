BrandFolderForge - Brand Directory Structure Generator

OVERVIEW
========
BrandFolderForge is a Python tool that creates a standardized directory structure
for managing brand assets and related materials. It automatically generates folders
for website content, marketing materials, brand assets, and operational documents.

REQUIREMENTS
===========
- Python 3.6 or higher
- No additional packages required (uses built-in modules only)

INSTALLATION
===========
1. Download the repository
2. Navigate to the folder in your terminal/command prompt
3. Run: python requirements.py

USAGE
=====
1. Open terminal/command prompt
2. Navigate to the script location
3. Run: python create_brand_directory.py
4. A new "Brand_Directory" folder will be created with the complete structure

USE CASES
========
- Setting up new client folders in agencies
- Organizing rebranding projects
- Creating consistent folder structures for marketing teams
- Standardizing asset management across organizations

The tool is particularly useful when:
- Starting work with a new client
- Setting up a new brand
- Reorganizing existing brand assets
- Establishing standard operating procedures
- Creating template structures for multiple brands
